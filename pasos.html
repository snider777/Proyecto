1. PARA CONVERTIR DE GEOGRÁFICAS A UTM:

En esta conversión se transforma una coordenada expresada en grados de latitud y longitud a coordenadas UTM (Universal Transverse Mercator), que se expresan en metros. Para realizar esta conversión, se le solicitará al usuario que ingrese la latitud y longitud del punto geográfico, indicando el hemisferio y zona UTM correspondientes para obtener las coordenadas proyectadas precisas.

2. PARA CONVERTIR DE UTM A GEOGRÁFICAS:

Este proceso permite transformar las coordenadas UTM (en metros) de vuelta a coordenadas geográficas en grados de latitud y longitud. El usuario deberá facilitar las coordenadas Easting, Northing, la zona UTM y el hemisferio en el que se encuentra el punto para que el sistema realice la conversión correcta a valores geográficos tradicionales.
